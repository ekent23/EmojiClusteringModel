# Emoji Clustering using K-Means

## Overview
This project applies K-means clustering to group emojis based on their features/similarity.

## Objective
To explore whether emojis naturally form meaningful clusters using unsupervised learning.

## Methods
- K-means clustering
- Python (NumPy, Pandas, Scikit-learn)
- Data preprocessing and feature extraction

## Results
- Emojis grouped into X clusters
- Observed patterns: (e.g., emotions grouped together, objects grouped together)

## Files
- K_means.ipynb: main analysis notebook
